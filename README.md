# Blockchain Lab

AI Agent - Codex를 활용해 블록체인을 단계별로 학습하고 구현하는 프로젝트입니다.

각 Stage별 학습목표는 /docs 디렉토리에 위치합니다.

## Version Choice

- Spring Boot: `3.5.14`
- Java: `17`
- Build: Gradle Wrapper


## Learning Rule

각 단계는 다음 순서로 진행합니다.

1. 개념 설명
2. 설계 문서 작성
3. 테스트 시나리오 정의
4. 코드 구현
5. 테스트
6. 이해 체크

실행:

```cmd
gradlew.bat bootRun
```

테스트:

```cmd
gradlew.bat test
```
